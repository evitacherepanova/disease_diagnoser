# Disease Diagnoser

Простая программа для предварительной диагностики заболеваний: Грипп или Ангина.

## О проекте

Это консольное приложение на Python, которое помогает определить вероятное заболевание на основе симптомов. Программа задает вопросы о симптомах и выдает предварительный диагноз с рекомендациями.

**Важно:** Проект создан в учебных целях и не заменяет консультацию врача.

## Функциональность

- Сбор информации о симптомах через консольный интерфейс
- Анализ симптомов и расчет вероятности заболеваний
- Вывод предварительного диагноза
- Рекомендации по дальнейшим действиям

## Требования

- Python 3.6 или выше
- Внешние зависимости не требуются

## Установка и запуск

```bash
# 1. Клонируйте репозиторий
git clone https://github.com/EvitaCherepanova/disease_diagnoser.git

# 2. Перейдите в папку проекта
cd disease_diagnoser

# 3. Создайте виртуальное окружение (рекомендуется)
python -m venv 

# 4. Активируйте виртуальное окружение
# Для Windows:
venv\Scripts\activate
# Для MacOS:
source venv/bin/activate

# 5. Запустите программу
python main.py

# 6. Если уже был запуск, сначала нужно удалить вр окружение
rmdir /s venv
rm -rf venv    # macOS/Linux
python -m venv venv # сделать новое
venv\Scripts\activate      # Windows
source venv/bin/activate   # macOS/Linux
